## **M07 — Reviews & Notifications**



---

### **F-07.1 Review Submission**

**Summary**

Verified purchasers submit star ratings and text reviews for products they have received. One review per customer per product.

**User Story**

As a customer who received a product, I want to leave a rating and review so that other shoppers can benefit from my experience.

**Flow — Happy Path**

1. Customer visits a product page or /account/orders/:orderId — 'Write a Review' prompt shown for delivered items.
2. Customer selects star rating (1–5), writes review text, submits.
3. POST /reviews creates review with status='pending' (awaiting moderation, F-07.2).
4. Customer sees 'Thank you for your review! It will appear once approved.'

**Flow — Edge Cases**

| Edge Case | Expected System Behaviour |
| :---- | :---- |
| Customer not a verified purchaser of that product | POST returns 403: 'You can only review products you have purchased.' |
| Duplicate review (same customer, same product) | 422: 'You have already reviewed this product.' Customer offered PATCH /reviews/:reviewId to edit instead. |
| Review contains profanity (auto-moderation) | If auto-mod enabled: status set to 'flagged' rather than 'pending'. Admin alerted. |

**Business Rules**

| BR \# | Rule |
| :---- | :---- |
| BR-01 | TC-24: Only customers with a delivered order containing the product may submit a review. Verified by purchase check on server. |
| BR-02 | One review per customer per product. Duplicate attempt returns 422 with link to edit existing review. |
| BR-03 | Reviews are not immediately visible. Status must be 'approved' by admin moderation (F-07.2) before appearing on product page. |
| BR-04 | TC-25: Review must appear on product page within 2 business days of submission (moderation SLA). |

**Input Validation Rules**

| Field | Type | Required | Validation Rules | Error Message Shown |
| :---- | :---- | :---- | :---- | :---- |
| productId | UUID | Yes | Must be a product the customer has received in a delivered order. | 403: 'You can only review products you have purchased.' |
| rating | Integer | Yes | 1–5 inclusive. | 'Please select a rating between 1 and 5 stars' |
| title | String | No | Max 150 chars. Plain text. Trimmed. | 'Review title must be 150 characters or fewer' |
| body | String | Yes | Min 10 chars, max 2000 chars. Plain text. HTML stripped. | 'Review must be between 10 and 2000 characters' |

**Permission Matrix**

| Method | Endpoint | Description | Guest | Customer | Ops | Content | Super Admin |
| :---- | :---- | :---- | :---- | :---- | :---- | :---- | :---- |
| **POST** | /api/v1/reviews | Submit review | ❌ | ✅ | ❌ | ❌ | ✅ |
| **GET** | /api/v1/reviews/me | Get customer's own reviews | ❌ | ✅ | ❌ | ❌ | ✅ |
| **PATCH** | /api/v1/reviews/:reviewId | Edit own review | ❌ | ⚠ | ❌ | ❌ | ✅ |
| **DELETE** | /api/v1/reviews/:reviewId | Delete own review | ❌ | ⚠ | ❌ | ✅ | ✅ |
| **POST** | /api/v1/reviews/:reviewId/helpful | Mark review as helpful | ❌ | ✅ | ❌ | ❌ | ✅ |

*⚠ Customer: own reviews only.*

**UI / Display Rules**

| Condition / State | UI Component | Display Text | Colour / Style | Available Action |
| :---- | :---- | :---- | :---- | :---- |
| Review pending | Customer's review in /account/orders | 'Your review is awaiting approval.' | Amber badge | 'Edit Review' link |
| Review approved | Visible on product page | Star rating + review text + 'Verified Purchase' badge | Standard card | 'Helpful?' upvote button |
| Review rejected | Customer's review in /account/orders | 'Your review was not approved. See our review guidelines.' | Red badge | 'Contact support' link |
| Edit review | Inline form pre-populated with existing content | — | Standard form | Save / Cancel |

**API Contract**

| Method | Endpoint | Description | Request Body / Params | Response |
| :---- | :---- | :---- | :---- | :---- |
| **POST** | /api/v1/reviews | Submit review | *{ productId, rating, title?, body }* | *201: { reviewId, status: 'pending' } / 403 / 422* |
| **GET** | /api/v1/reviews/me | Own reviews | *?page=* | *200: { reviews[{ id, product, rating, status, createdAt }] }* |
| **PATCH** | /api/v1/reviews/:reviewId | Edit review | *{ rating?, title?, body? }* | *200: { review } — resets status to 'pending'* |
| **DELETE** | /api/v1/reviews/:reviewId | Delete review | *—* | *200: { message }* |
| **POST** | /api/v1/reviews/:reviewId/helpful | Upvote review | *—* | *200: { helpfulCount }* |

**TC References:** TC-24 (verified purchase requirement), TC-25 (2 business day moderation SLA)
